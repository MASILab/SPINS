
# angular-gyrus

## Overview

The right angular gyrus is a multimodal association region located in the posterior part of the inferior parietal lobule, bordering the superior temporal and occipital cortices, and corresponding roughly to Brodmann area 39. It integrates visual, auditory, and somatosensory inputs and is implicated in higher-order cognitive functions including language processing (particularly semantic processing and reading), number processing and arithmetic, spatial cognition and attention, theory of mind, and aspects of memory retrieval. Structurally, it lies at the junction of the intraparietal sulcus, the superior temporal sulcus, and the lateral occipital cortex, with extensive connectivity to frontal (e.g., inferior frontal gyrus), temporal (e.g., middle temporal gyrus), and limbic structures, forming part of the default mode and fronto-parietal control networks. Disruption of the right angular gyrus has been associated with deficits in spatial attention, neglect syndromes, number processing impairments, and altered self–other distinction.
