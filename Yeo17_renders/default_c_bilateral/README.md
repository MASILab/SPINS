
# Default_C

## Overview

The Bilateral Default_C region in the Yeo-17 functional atlas is a subdivision of the default mode network (DMN), a large-scale intrinsic connectivity system most active during rest and internally directed cognition. Default_C typically encompasses midline and adjacent bilateral cortical territories that participate in self-referential processing, autobiographical memory, mental scene construction, and aspects of social cognition. Functionally, this subdivision is characterized by strong low-frequency synchronous activity with other DMN components and relative deactivation during externally focused, attentionally demanding tasks. Anatomically, it overlaps with portions of medial parietal and medial prefrontal cortices and is interconnected with limbic and temporal association areas, supporting integration of mnemonic, emotional, and conceptual information into coherent internal models of the self and environment. There is no direct Wikipedia page for the “Bilateral Default_C” region; a closely related structure is the default mode network: https://en.wikipedia.org/wiki/Default_mode_network
