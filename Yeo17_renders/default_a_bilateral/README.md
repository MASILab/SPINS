
# Default_A

## Overview

The Bilateral Default_A region in the Yeo-17 functional atlas corresponds to a core subdivision of the default mode network (DMN), a large-scale brain system showing high activity during rest and internally focused cognition. This subdivision is bilaterally distributed and typically encompasses medial prefrontal cortex, posterior cingulate cortex/precuneus, and angular or inferior parietal regions, which are implicated in self-referential processing, autobiographical memory, mind-wandering, and aspects of social cognition. Functionally, Default_A is thought to support internally directed thought and integration of information about the self and others over time, in contrast to networks involved in externally oriented attention or sensorimotor processing.
